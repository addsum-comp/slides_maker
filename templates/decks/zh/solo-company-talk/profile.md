# 一人公司演讲模板（中文）

## 模板文件

`template.pptx`

## 概述

12 页、16:9 的演讲 / 主题分享套牒，主题「INDIE × AI」：一个人 + AI 当全栈团队，如何搭起一家小公司。视觉体系大胆、冲击力强、满图铺陈、适合舞台演讲：开场镇场、概念、趋势、时代转折、真实案例、方法论、风险与现实、收尾。文件较大（含大量整屏图像）。英文版见 `en/solo-company-talk`。

当用户想做一份有煽动力、看点密集的中文演讲 deck（主题分享、路演开场、观点输出）时，用它作为起点。

## 结构

- 页面尺寸 10.00 x 5.62 英寸（16:9）。页数 12。版式 11 套。
- 字体：标题 Arial Black、正文 Arial；中文经系统回退到苹方渲染（macOS 上正常）。
- 每页多为整屏大图 + 少量强文字，观点一页一击。

## 适合

- 舞台演讲、主题分享、路演开场。
- 观点驱动、看点密集、以气势取胜的内容。

## 复用规则

- 保留节奏：强开场 → 概念 → 趋势 → 转折 → 案例 → 方法 → 风险 → 收尾，一页一个观点。
- 复用大胆的满图视觉，替换成用户的观点、案例与配图。
- 若要跨环境稳定渲染中文，可把字体显式设为 Hiragino Sans GB（当前依赖系统回退）。

## Agent 提示

1. 构建前先跑 `scripts/inspect_template.py templates/decks/zh/solo-company-talk/template.pptx`。
2. 用 `deckkit.open_template()` 基于本套牒构建。
3. 文件较大，整屏图片是核心，替换配图时注意压缩。
4. 把模板当作视觉系统，而非内容来源。
