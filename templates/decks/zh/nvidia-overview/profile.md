# 公司概览模板（中文）

## 模板文件

`template.pptx`

## 概述

16 页、16:9 的企业概览 / 公司介绍套牒（示例内容为 NVIDIA）。视觉体系自信、现代、深色科技风：强封面、身份陈述、发展故事、产品支柱、数字页、增长曲线、收尾行动号召。这是英文版 `en/nvidia-overview` 的中文版本，正文已译成自然中文，产品名与公司名按原样保留。

当用户想做一份精致的「我们是谁 / 造什么 / 为什么重要」中文 deck（公司概览、融资骨架、投资人介绍、产品矩阵巡览、招聘品牌页）时，用它作为起点。

## 结构

- 页面尺寸 10.00 x 5.62 英寸（16:9）。
- 页数 16。版式 11 套。
- 字体：正文走 PingFang SC；产品 / 公司名、数字、货币、日期、URL 保留原拉丁字体。

## 适合

- 公司 / 创业公司概览与企业介绍。
- 投资人与合作伙伴的融资骨架。
- 产品矩阵与平台巡览、招聘品牌页。

## 复用规则

- 保留结构：强封面 → 身份 → 故事 → 造什么 → 数字 → 产品支柱 → 价值 → 增长曲线 → 行动号召。
- 产品名、公司名、人名、数字、货币保持原样，别硬译。
- 复用自信的现代深色视觉与节奏，替换成用户真实素材。
- CJK 字体沿用 PingFang SC，避免中文豆腐块。

## Agent 提示

1. 构建前先跑 `scripts/inspect_template.py templates/decks/zh/nvidia-overview/template.pptx`。
2. 用 `deckkit.open_template()` 基于本套牒构建。
3. 从源套牒提取配色、字体与重复元素，别硬编码新颜色。
4. 把模板当作视觉系统，而非内容来源。
