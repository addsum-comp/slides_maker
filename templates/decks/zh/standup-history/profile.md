# 脱口秀简史模板（中文）

## 模板文件

`template.pptx`

## 概述

12 页、16:9 的历史 / 演变叙事套牒，以「脱口秀简史（1866 → 今天）」为例，按时代逐段推进：定义、演讲巡回、杂耍时代、麦克风时代、审查、黄金年代、俱乐部热潮、全球化、流媒体、中国、结语。视觉体系是编年史式的，分节小标题带英文副题（如「定义 · WHAT IS STAND-UP」）。英文版见 `en/standup-history`。

当用户想做一份「某事物如何一路演变到今天」的中文叙事 deck（历史脉络、行业演进、文化题材、时间线故事）时，用它作为起点。

## 结构

- 页面尺寸 10.00 x 5.62 英寸（16:9）。页数 12。版式 11 套。
- 字体：正文 Hiragino Sans GB；英文副题与数字用 Arial / Arial Black。

## 适合

- 历史脉络、演变简史、时间线叙事。
- 文化 / 行业题材的「从起源到今天」讲解。

## 复用规则

- 保留编年节奏：定义 → 分时代 → 当下 → 结语，一个时代一页。
- 复用分节小标题（中文主标题 + 英文副题）的排版，替换成用户的题材与年代。
- CJK 字体沿用 Hiragino Sans GB，避免中文豆腐块。

## Agent 提示

1. 构建前先跑 `scripts/inspect_template.py templates/decks/zh/standup-history/template.pptx`。
2. 用 `deckkit.open_template()` 基于本套牒构建。
3. 把模板当作视觉系统，而非内容来源。
