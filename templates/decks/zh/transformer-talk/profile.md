# Transformer 架构精读模板（中文）

## 模板文件

`template.pptx`

## 概述

15 页、16:9 的论文精读 / 组会汇报套牒，围绕 Vaswani 等人的 `Attention Is All You Need` 展开。它适合作为“组会 / 论文汇报”方向的主展示模板：从背景问题、核心主张、模型架构、注意力公式、多头注意力、位置编码，到实验结果与局限，完整覆盖一篇经典论文汇报的讲述路径。

这套模板不是空占位，而是一份带真实内容的示例 deck。复用时保留视觉系统、节奏和论文汇报结构，替换为用户自己的论文内容。

## 结构

- 页面尺寸 10 x 5.625 英寸（16:9）。
- 页数 15。
- 语言：中文为主，论文标题、机构、术语、指标和公式按原文保留。
- 示例内容：`Attention Is All You Need`，NIPS 2017 / arXiv:1706.03762。

## 适合

- 论文精读、组会汇报、方法综述。
- 经典论文 / 新论文的结构化讲解。
- 需要展示论文 figure、公式、实验结果、局限和启发的技术汇报。

## 复用规则

- 保留讲述节奏：背景问题 → 核心主张 → 架构拆解 → 关键机制 → 实验结果 → 局限与启发。
- 复用的是视觉系统和页型，不是 Transformer 的具体结论；除非用户的新主题就是 Transformer，否则替换所有论文事实、数字、图表和公式。
- 论文 figure、表格和公式必须来自用户材料或重新排版，不能把示例里的 Transformer 内容混进新主题。
- 适合作为中文科研学生的默认学术汇报起点。

## Agent 提示

1. 构建前先跑 `scripts/inspect_template.py templates/decks/zh/transformer-talk/template.pptx`。
2. 用 `deckkit.open_template()` 基于本套牒构建。
3. 保留页面密度、编号节奏、公式/figure/结果页的组织方式。
4. 讲稿型汇报建议继续写入 speaker notes，让用户能直接开讲。
