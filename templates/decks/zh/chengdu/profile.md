# 成都旅行指南模板（中文）

## 模板文件

`template.pptx`

## 概述

12 页、16:9 的旅行 / 城市指南套牒，围绕成都展开。视觉体系温暖、编辑感强、以大图为主，节奏像一段被引导的故事：封面、为何值得、分节页、要点页，再到一个收尾的情绪页。这是英文版 `en/chengdu` 的中文版本，正文已译成自然中文；主标题使用中文，标题下方的解释性副标题使用英文。

当用户想做一份关于某地、某文化、某活动或生活方式的鲜活但克制的中文 deck 时，用它作为起点。

## 结构

- 页面尺寸 13.33 x 7.50 英寸（16:9）。
- 页数 12。版式 11 套（标题页、分节页、两栏、图注页等）。
- 字体：无衬线走 Hiragino Sans GB，衬线（对应原 Georgia）走 Songti SC；数字与拉丁字保留原拉丁字体。

## 适合

- 城市 / 目的地指南。
- 文化、美食、旅游、生活方式类解读。
- 讲氛围但仍要保证可读性的对外内容。

## 复用规则

- 保留节奏：强封面、短分节、图片主导的要点页、克制的说明文字。
- 复用的是视觉系统与节奏，不是成都的具体内容，替换成用户真实素材。
- 文字要短，一页由一张大图或一个场景来承载效果最好。
- 主标题用中文，标题下方的解释性副标题用英文；正文说明继续使用中文。
- CJK 字体沿用 Hiragino Sans GB / Songti SC，避免中文渲染成豆腐块。

## Agent 提示

1. 构建前先跑 `scripts/inspect_template.py templates/decks/zh/chengdu/template.pptx`。
2. 用 `deckkit.open_template()` 基于本套牒构建。
3. 从源套牒提取配色、字体与重复元素，别硬编码新颜色。
4. 把模板当作视觉系统，而非内容来源。
