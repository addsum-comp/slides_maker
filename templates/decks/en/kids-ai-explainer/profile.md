# AI for Kids Template (English)

## Template File

`template.pptx`

## Summary

A 16-slide, 16:9 kid-friendly explainer deck that teaches "AI" to children. The visual system is playful, big-type, and colourful, paced in numbered parts (Part 1 · Meet AI, Part 2 · It learns …). Each part pairs a big title slide with a "one line / an example" slide, closing with an "Imagine" and a recap. Chinese version: `zh/kids-ai-explainer`.

Use this when the user wants a light, easy-to-grasp explainer for kids or a general audience.

## Inspected Structure

- Slide size: 10.00 x 5.62 in (16:9). Slides: 16. Layouts: 11.
- Font: Trebuchet MS throughout — a friendly sans that suits large, playful type.

## Use It For

- Kids / general-audience explainers; any "make one concept simple" topic.
- Sectioned teaching, classroom, light knowledge decks.

## Reuse Rules

- Keep the part rhythm: each part = a big title slide + one takeaway slide; keep language spoken and sentences short.
- Reuse the playful big-type visual; replace with the user's topic and examples.
