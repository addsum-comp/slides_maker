# Solo Company Talk Template (English)

## Template File

`template.pptx`

## Summary

A 12-slide, 16:9 talk / keynote deck on the theme "INDIE × AI": how one person plus AI as a full-stack team can build a company. The visual system is bold, high-impact, and full-bleed image-led, made for the stage: the hook, the idea, the trend, the turn, real cases, the playbook, risks & reality, close. Large file (lots of full-screen imagery). Chinese version: `zh/solo-company-talk`.

Use this when the user wants a punchy, high-energy talk deck (keynote, pitch opener, opinion piece).

## Inspected Structure

- Slide size: 10.00 x 5.62 in (16:9). Slides: 12. Layouts: 11.
- Fonts: Arial Black (display) + Arial (body).
- Mostly full-screen images with a few strong words — one idea per slide.

## Use It For

- Keynotes, talks, pitch openers.
- Opinion-driven, high-energy, atmosphere-first content.

## Reuse Rules

- Keep the rhythm: strong hook → idea → trend → turn → cases → playbook → risks → close; one idea per slide.
- Reuse the bold full-bleed visual; replace with the user's argument, cases and imagery.

## Agent Notes

1. Run `scripts/inspect_template.py templates/decks/en/solo-company-talk/template.pptx` before building.
2. Use `deckkit.open_template()` when building on this preset.
3. Large file — full-screen images are the core; recompress when replacing imagery.
4. Treat the template as a visual system, not as source content.
